本页面将引用jQuery的两个插件。注意由于在shim中配置了依赖，因此依赖数组中可以不加"jquery"，RequireJS会自动处理好依赖。

如果几乎所有的模块都依赖于jQuery，则建议使用map来配置依赖。具体 查看此处


        requirejs(['../common'], function (common) {

            // 由于在shim中配置了依赖，因此依赖数组中可以不加"jquery"
            requirejs(["jquery.alpha", "jquery.beta"], function () {
                $(function () {
                    $('body').alpha().beta();
                });
            });

        });
